# Note Trainer

Native iOS app for music-reading practice. The active product is the SwiftUI app in `NoteTrainer/`.

## Repo layout

- `NoteTrainer/`: native SwiftUI iOS app
- `NoteTrainerTests/`: unit tests
- `legacy-web/`: archived web prototype kept as a migration reference
- `AGENT_NOTES.md`: working repo summary
- `IOS_BLUEPRINT.md`: architecture/migration plan

## Development

1. Open `NoteTrainer.xcodeproj`
2. Build the `NoteTrainer` scheme
